Estrutura HTML

!DOCTYPE html
Define o tipo de documento com HTML 5.
HTML
Define um documento HTML.
HEAD
Container para metadados, título da página, links...
BODY
Corpo do documento.

Tags Obsoletas

BIG
Foi uma tag usada para aumentar o tamanho da fonte nos navegadores,
seu uso não é mais recomendado pois a uma função ainda melhor no CSS a font-size.
FRAME
Era usado para criação de quadros para melhorar a acessibilidade,
hoje em dia esse recurso pode causar problemas de acessibilidade por conta do CSS.
STRIKE
Era usado para deixar o texto riscado para indicar que tal texto deve ser ignorado,
deixou de ser usado por problemas de compatibilidade entre os navegadores ocasionando bugs,
para o substituto dessa tag temos o DEL.
FONT
Era usado para modificar a fonte do documento, mudando sua cor, tamanho e tipo.
Não recomendado o seu uso já que tenho as folhas de estilo CSS.
